Praticando com o Git e Github

O que é o GIT?

~

O git é um programa de controle de alterações de código. Ele possibilita a vizualização do histórico de alterações e aceitações de versões. O git cria um repositório local.

O que é o Github

O Git Hub é uma espécie de rede social para desenvolvedores, ela possibilita a publicação de seus projetos. O git também é um repositório remoto.
